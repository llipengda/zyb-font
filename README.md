# zyb-font -- 生成你的手写字体

## 部署

创建 `Python` 虚拟环境

- Linux

```bash
python3 -m venv venv
```

- Windows

```pwsh
python -m venv venv
```

激活虚拟环境

- Linux

```bash
source venv/bin/activate
```

- Windows(PowerShell)

```pwsh
.\venv\Scripts\activate.ps1
```

- Windows(CMD)

```cmd
.\venv\Scripts\activate.bat
```

安装依赖

```bash
pip install -r requirements.txt
```
